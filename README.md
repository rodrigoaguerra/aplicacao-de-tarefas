# Desafio 2 - Aplicação de Tarefas com Rotas, Estado, Efeito e Contexto

### Resumo do Projeto

Crie uma aplicação React para gerenciar tarefas, utilizando rotas, estado, efeito e contexto.

#### Introdução

Neste exercício, você desenvolverá uma aplicação mais complexa para gerenciar uma lista de tarefas (to-do list). Este projeto é ideal para praticar o uso de hooks, gerenciamento de estado global com Context API, e navegação entre diferentes páginas usando React Router.

### Especificações

Você deverá criar uma aplicação React que permita ao usuário adicionar, visualizar, editar e excluir tarefas. A aplicação deve possuir múltiplas páginas, utilizando React Router para navegação. Utilize Context API para gerenciar o estado global das tarefas.

### Requisitos
1. **Configuração do Ambiente**: Configure um novo projeto React utilizando o Create React App.

2. **React Router**: Instale e configure o React Router para criar pelo menos duas páginas: Home e Adicionar Tarefa.

3. **Context API**: Crie um contexto para gerenciar o estado global das tarefas.

4. **Hooks**: Utilize os hooks useState e useEffect para gerenciar o estado e os efeitos colaterais.

5. **CRUD de Tarefas**: Implemente funcionalidades para adicionar, visualizar, editar e excluir tarefas.

6. **Persistência**: Utilize localStorage para persistir as tarefas entre sessões.

### Execução do código
1. **Abra um terminal** em sua máquina
2. **Clone o repositório** para sua máquina local:
```bash
git clone https://github.com/rodrigoaguerra/aplicacao-de-tarefas.git
```
3. **Acesse o diretório do projeto**:
```bash
cd aplicacao-de-tarefas
```
4. **Instale as dependências do projeto**:
```bash
npm install
```
5. **Inicie a aplicação**:
```bash
npm start
```
6. **Acesse o aplicativo** pelo navegador no endereço:
[http://localhost:3000](http://localhost:3000)
